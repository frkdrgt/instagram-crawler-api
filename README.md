# Instagram Crawler API

Crawl Instagram images ande videos using Puppeter library. It creates a json file.
Use responsibly.

## Installation
> git clone https://github.com/frkdrgt/instagram-crawler-api.git    
> npm install

## Usage
> node index.js     
> localhost:5000/user/instagram
